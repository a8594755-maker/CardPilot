# Security Policy

## Supported Versions
Security fixes are applied to the active `main` branch and the latest release tag.

## Reporting a Vulnerability
Preferred channels:
1. GitHub Security Advisory (private report)
2. Email: `security@cardpilot.app`

If neither is available, open a GitHub issue and add the `security` label, but do not post exploit details publicly.

Include:
- Affected commit/tag
- Reproduction steps
- Impact assessment
- Logs or stack traces (sanitized)

## Response Targets
- Acknowledgement: within 3 business days
- Initial triage: within 7 business days
- Remediation timeline shared after triage

## Disclosure
Please allow maintainers time to triage and patch before public disclosure.
